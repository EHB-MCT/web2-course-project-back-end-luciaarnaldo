# Recipe App 
Recipe App is a web application that allows users to browse, filter and view recipes.
Users can register and log in, apply filters by ingredients, time and type of food, and view detailed recipe information including images and preparation steps.



## Up & running 
- Open a terminal in the back-end and run `node db/seed.js`and then `npm start`
- Open a terminal in the front-end and run `npm run dev`
- Search http://localhost:5173/
- Login to the test account using these credentials, or you can register a new user:
